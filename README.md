# Proyecto-Java: Gestor de Librería Personal
**Descripción General**
Aplicación en Java que permite gestionar tu biblioteca personal de libros.

Busca información en la API pública de OpenLibrary, permite marcar el estado de lectura, puntuar libros, guardar favoritos y ver estadísticas de lectura.

API base: https://openlibrary.org. Sin autenticación y completamente gratuita y pública.
